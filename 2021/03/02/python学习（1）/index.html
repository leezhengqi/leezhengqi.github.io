python之爱在心口难开

人生苦短，我用Python

记得Python大火的那段时间，到处都是催生荷尔蒙激素的宣传语，冲动之下，我入“坑”了。
鸡血了一阵子，根据课程的进度，打卡学习，不可否认，也学了一些东西，但是最初的激情随着时间的推移，开始慢慢消退。
其中一个关键因素是：用不到。如今想来，最容易导致学习半途而废的，恐怕非它莫属了。试想，每天只进不出，是啥感觉？行走的“貔貅”吗！

实际上，在这背后，更深层次的原因是没有反馈。不管什么事情，若想持续下去，即时反馈既重要又必要，因为这能让你意识到：哦，这个玩意用在这个地方，蛮不错的，我得好好学。总之，无论是自身获得的成就感，还是来自外部的认可，都会是你坚持下去的动力。
当初没能认识到这一点，所以我的第一次Python之旅，还没怎么开始，就宣告结束了。

时光荏苒，斗转星移。

编程这玩意就像勾子一样，深深地埋在了我的心头，说不定什么时候就会被“勾引”出来。
这次被“勾引”，再次启动python学习，缘起于老胥不经意间说起的一句话。

老胥何许人也？
之前文章里提到的编程大神，是也！
他说，抛开什么大数据，人工智能的不谈，就说日常办公这方面。如今Wps用的人越来越多，而VBA虽然在office里用起来很方便，但wps对它却不太友好。而python就没有这样的限制了，叫他万能胶水语言，一点也不为过。
而且，python学起来不难，因为它的语法比较接近自然语言，只要你逻辑思路够清楚，又不对英语有很大反感，任何人都可以学会。

说实话，最后这句给我吃了个定心丸。于是，开始蠢蠢欲动，摩拳擦掌。
但真正地再次踏上python学习之旅，还是随后出现的一件事，引发的。
欲知后事如何，请听下回分解。
